# Multiverso
Desafio: O Multiverso Precisa de Organização! O Multiverso Nerd entrou em colapso. Os portais entre universos se abriram e personagens de mundos diferentes estão misturados em uma base de dados completamente confusa: nomes duplicados, franquias desorganizadas, campos escritos de forma diferente e até espécies misteriosas mal registradas!
